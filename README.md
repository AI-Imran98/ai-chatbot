# 🤖 AI Chatbot

An intelligent AI chatbot powered by Google Gemini API!

## ✨ Features
- Real-time AI conversation
- Chat history memory
- Clear chat option
- Clean, professional UI
- Built with Python & Streamlit

## 🛠️ Tech Stack
- Python 3
- Google Gemini 2.0 Flash API
- Streamlit
- python-dotenv

## 🚀 How to Run

### 1. Clone the repository
git clone https://github.com/AI-Imran98/ai-chatbot
cd ai-chatbot

### 2. Install dependencies
pip install -r requirements.txt

### 3. Setup API Key
Create a .env file:
GEMINI_API_KEY=your_api_key_here

### 4. Run the app
streamlit run app.py
